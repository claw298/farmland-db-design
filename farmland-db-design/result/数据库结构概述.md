## 8. 业务逻辑说明

### 8.1 用户-角色-权限体系

```
user ─── user_role ─── role ─── role_permission ─── permission
   │
   └── user_organization ─── organization
```

### 8.2 基地-地块体系

```
organization ── user_organization ── user
                                    └─ base_farmer ── base
                                                        ├─ base_plot ── plot
                                                        └─ base_agent ── user (agent)
```

### 8.3 生长季-作物-地块体系

```
growing_season ── growing_season_crop ── crop
              └── growing_season_plot ── plot
              └── growing_season_growth_stage ── growth_stage
```

### 8.4 监测数据体系

```
plot ── weather_daily/weather_hour (气象)
     ── soil_test (土壤检测)
     ── sensor_plot ── sensor (传感器)
     ── remote_sensing_image (遥感)
     ── alert (预警)
     ── report (报表)
     ── farm_activity (农事活动)
```

## 9. 导入注意事项

导入SQL文件时必须按以下顺序执行：

1. **第一层**: organization, user, role, permission, crop, growth_stage, alert_level, alert_type, report_type, sensor_type, soil_test_items, soil_test_templates, soil_items_templates, pest_disease_type, activity_type
