## 六、修复建议

### 6.1 立即修复 (严重问题)

```sql
-- 1. base_plot: 添加唯一约束，确保地块只属于一个基地
ALTER TABLE base_plot
ADD CONSTRAINT uk_base_plot_plot UNIQUE (plot_id);

-- 2. growing_season: 添加基地外键和唯一约束
ALTER TABLE growing_season
ADD COLUMN base_id BIGINT COMMENT '基地ID',
ADD CONSTRAINT fk_growing_season_base FOREIGN KEY (base_id) REFERENCES base(id),
ADD CONSTRAINT uk_growing_season_name UNIQUE (base_id, name);

-- 3. user_role: 添加约束确保唯一主角色
ALTER TABLE user_role
ADD CONSTRAINT uk_user_primary_role UNIQUE (user_id, is_primary);
```

### 6.2 建议添加的表

```sql
-- 操作日志表
CREATE TABLE operation_log (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT COMMENT '操作用户',
    action VARCHAR(100) COMMENT '操作类型',
    target_table VARCHAR(100) COMMENT '目标表',
    target_id BIGINT COMMENT '目标ID',
    detail JSON COMMENT '操作详情',
    ip VARCHAR(50) COMMENT 'IP地址',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 传感器数据表
CREATE TABLE sensor_data (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    sensor_id BIGINT NOT NULL COMMENT '传感器ID',
    time DATETIME NOT NULL COMMENT '采集时间',
    value DECIMAL(10,2) COMMENT '采集值',
    quality TINYINT COMMENT '数据质量',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_sensor_time (sensor_id, time)
);

-- 系统配置表
CREATE TABLE system_config (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    config_key VARCHAR(100) NOT NULL UNIQUE COMMENT '配置键',
    config_value TEXT COMMENT '配置值',
    description VARCHAR(255) COMMENT '配置描述',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME ON UPDATE CURRENT_TIMESTAMP
);
```

### 6.3 CHECK 约束添加

```sql
-- 地块面积必须大于0
ALTER TABLE plot ADD CONSTRAINT chk_area_positive CHECK (area_mu > 0);

-- 用户ID必须大于0
ALTER TABLE user ADD CONSTRAINT chk_user_id_positive CHECK (user_id > 0);

-- 传感器设备ID必须大于0
ALTER TABLE sensor ADD CONSTRAINT chk_sensor_id_positive CHECK (sensor_id > 0);
```
